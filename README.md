### 项目介绍

此项目为果园云平台项目打数据展示前端模块

- 设计效果图地址:[传送门](https://app.mockplus.cn/app/GAgvrQ2RT/develop/design/POiWHQTHA)
- 构建工具:Vite
- 开发语言:TypeScript
- 主框架:React + styled-components + echarts +Axios

### 快速开始
#### 安装依赖
    
```bash
# 使用 npm
npm install
# 使用 yarn
yarn
```

#### 开发预览

```bash
npm run dev
```

#### 打包

```bash
npm run build
```


#### 项目结构

```bash
├── src
│   ├── main.tsx # 入口文件
│   ├── App.tsx # 根组件
│   ├── config.ts # 全局配置参数 
│   ├── components # 组件
│   ├── api # 接口
│   ├── assert # 静态资源
│   ├── global-style # 全局样式
│   ├── utils # 工具函数
```
